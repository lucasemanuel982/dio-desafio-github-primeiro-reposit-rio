Comandos 

DIR - aparece em todas as pastas. 
CD - navega entre as pastas
cd.. - volta para a pasta anterior.
cls - apaga tudo do terminal. - Atalho= CTRL + L
TAB - a tecla TAB auto completa a determinada pasta ou comando. 
mkdir - criar uma pasta. 
ls - lista os arquivos 
echo > - cria um arquivo .
pwd - mostra o caminho que está sendo executado. 
del - apaga os arquivo da pasta
rmdir  - apaga o diretório/pasta
CTRL + l -  Limpa




git status - verifica o status do arquivo, se está untracked, modified, staged…
git init - iniciar o repositório do git
git add - mover arquivos e dá início aos primeiros comandos.
git commit - criar um git commit
